# **CurioCity アプリケーション仕様書**

## 1. プロジェクト概要

* **プロジェクト名**：CurioCity
* **現在のバージョン**：0.0.0
* **概要**：
  ユーザーが幻想的な架空都市や実在都市を旅しながら、雑学を学べるWebアプリ。
  最初に降り立つ地点はランダムに決定され、地点ごとに雑学と雰囲気に合った背景が表示される。
  芸術作品のようなUIを目指し、毎回の訪問で異なる体験を提供する。

---

## 2. 技術スタック（実装済み）

* **フロントエンド**：React 19.1.0 + TypeScript + Vite 7.0.4
* **UI フレームワーク**：Emotion 11.14.0 + styled-components + Framer Motion 12.23.12
* **地図システム**：Leaflet 1.9.4 + React Leaflet 5.0.0
* **ホスティング**：Firebase Hosting
* **データストア**：静的JSON（事前登録データ）
* **画像管理**：動的取得（Unsplash/Pexels API + フォールバック）
* **パフォーマンス監視**：Web Vitals 5.1.0 + カスタム監視システム
* **外部API**（実装済み）：
  * Unsplash API（フリー画像取得）
  * Pexels API（フリー画像取得）
  * 複数のフォールバック画像サービス

---

## 3. コンテンツ仕様（実装状況）

* **現在の雑学データ数**：実装済み（JSONファイル形式）
* **データ構造**：
  * `id`: 一意識別子
  * `title`: 雑学タイトル（メイン表示）
  * `short`: 短文説明（1〜2行）
  * `detail`: 詳細解説（約300文字）
  * `tags`: 雰囲気タグ（emotion/setting/palette）
  * `coords`: 位置情報（緯度経度）- 実装済み
  * `images`: 関連画像配列（現在は動的取得）

* **地点データ**：
  * `id`: 地点識別子
  * `name`: 日本語地点名
  * `nameEn`: 英語地点名  
  * `coords`: 座標情報
  * `type`: "real" | "fictional"（実在/架空）
  * `atmosphere`: 雰囲気タグ配列
  * `description`: 地点説明

* **画像システム**：タグベース動的選択（API連携済み）
